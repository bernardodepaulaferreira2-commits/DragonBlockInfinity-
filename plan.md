# Plano para DragonBlockInfinity

## Visão Geral
DragonBlockInfinity é um mod para Minecraft que traz elementos do universo Dragon Ball para o jogo, incluindo entidades, itens, blocos e mecânicas de combate.

## Objetivos Atuais
- Implementar entidades de dinossauros (já em andamento)
- Adicionar itens e blocos relacionados a Dragon Ball
- Criar mecânicas de Ki e transformações
- Desenvolver sistema de combate aprimorado

## Próximos Passos
1. **Entidades**
   - Completar modelos e texturas para dinossauros
   - Adicionar comportamentos de IA
   - Implementar drops e spawns

2. **Itens e Blocos**
   - Dragon Balls funcionais
   - Blocos de Namek
   - Itens de comida (Senzu, carne de dino)

3. **Mecânicas de Jogo**
   - Sistema de Ki
   - Transformações de personagens
   - Combate corpo a corpo aprimorado

4. **UI e Menus**
   - Menu de personagem
   - Interface para Dragon Balls

## Cronograma
- Semana 1: Finalizar entidades de dinossauros
- Semana 2: Implementar itens básicos
- Semana 3: Sistema de Ki inicial
- Semana 4: Testes e balanceamento
